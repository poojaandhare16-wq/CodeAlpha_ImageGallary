📸 CodeAlpha_ImageGallary

📌 Description
This is a simple Image Gallery web app built using HTML, CSS and JavaScript.
It allows users to view images, filter by category, and open images in a lightbox.

🚀 Features
* Image filtering (Nature, City, Animals)
* Lightbox view
* Next/Previous navigation
* Responsive design

 🛠️ Technologies Used
* HTML
* CSS
* JavaScript
 ▶️ How to Run

Open `index.html` in browser.
